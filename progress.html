Lead Capture and CRM Plumbing

Unified lead endpoint
  • Forms post to a single API that normalizes incoming fields.
  • Hidden UTM parameters are auto‑captured for attribution.
  • Property context (address, price, beds/baths/sqft, lat/lng) is attached when available.
Adapters

A pluggable adapter pattern enables swapping the CRM later (HubSpot/Salesforce/Follow Up Boss). Development uses a safe console adapter.

  • Configured EmailJS with Public Key (env), Service service_onh65w5, and Buyer/Seller templates.
  • Both the brokerage and the client receive emails upon submission (notification + confirmation).
  • Graceful fallback if keys are missing so the UX is never blocked.

Automated Daily Blog (No Manual Work)

The site includes a daily blog aggregator that pulls trusted real‑estate news sources and displays fresh posts automatically.

  • Curated sources (NAR, Redfin News, Zillow Porchlight, HousingWire, MBA).
  • Smart media extraction for cover images and YouTube thumbnails.
  • Duplicate filtering, sorting by date, and a one‑day cache to keep content fresh.
  • Exposed via an API endpoint and rendered on the Blog pages.
Result: The blog updates itself daily without manual edits.
